# opencode-prompt-engineering

Tools for reviewing, analyzing, refining, and optimizing LLM prompts for clarity, precision, goal effectiveness, and token efficiency

## Installation

```bash
./install.sh prompt-engineering
```

## Commands

- `/apply-prompt-feedback-prompt-engineering`
- `/compress-prompt-prompt-engineering`
- `/optimize-prompt-goal-prompt-engineering`
- `/optimize-prompt-precision-prompt-engineering`
- `/optimize-prompt-token-efficiency-prompt-engineering`
- `/review-prompt-prompt-engineering`

## Agents

- `prompt-compression-verifier-prompt-engineering`
- `prompt-feedback-verifier-prompt-engineering`
- `prompt-goal-verifier-prompt-engineering`
- `prompt-precision-verifier-prompt-engineering`
- `prompt-reviewer-prompt-engineering`
- `prompt-token-efficiency-verifier-prompt-engineering`
