# Prompt Engineering

Tools for reviewing, analyzing, refining, and optimizing LLM prompts for clarity, precision, goal effectiveness, and token efficiency.

## Commands

- `/apply-prompt-feedback` - Apply feedback to improve a prompt
- `/compress-prompt` - Compress a prompt to reduce token usage
- `/optimize-prompt-goal` - Optimize prompt for goal effectiveness
- `/optimize-prompt-precision` - Optimize prompt for precision
- `/optimize-prompt-token-efficiency` - Optimize prompt for token efficiency
- `/review-prompt` - Review and analyze a prompt

## Agents

- `prompt-compression-verifier` - Verify prompt compression quality
- `prompt-feedback-verifier` - Verify prompt feedback application
- `prompt-goal-verifier` - Verify prompt goal alignment
- `prompt-precision-verifier` - Verify prompt precision
- `prompt-reviewer` - Review prompts for quality
- `prompt-token-efficiency-verifier` - Verify token efficiency

## Installation

```bash
./install.sh prompt-engineering
```
