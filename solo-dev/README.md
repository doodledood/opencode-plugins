# opencode-solo-dev

Toolkit for solo developers to build, manage, and grow their business - customer profiling, brand guidelines, design system, SEO strategy, X/Twitter growth strategy, content creation, and more

## Installation

```bash
./install.sh solo-dev
```

## Commands

- `/audit-ux-solo-dev`
- `/craft-author-voice-solo-dev`
- `/define-brand-guidelines-solo-dev`
- `/define-customer-profile-solo-dev`
- `/define-design-guidelines-solo-dev`
- `/define-seo-strategy-solo-dev`
- `/define-x-strategy-solo-dev`
- `/write-as-me-solo-dev`

## Agents
